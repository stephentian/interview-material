# DP 动态规划

## 爬楼梯

地址：[70.爬楼梯](https://leetcode.cn/problems/climbing-stairs/)

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

分析：

最后一步上楼顶可能有两种方式，走一个台阶或者两个台阶。所以 最后一步的方法 等于前一步 加上 前两步的方法。

```js
var climbStairs = function(n) {
	// 1 = 1
  // 2 =
	// 1 + 1
	// 2
	// 3 = 
	// 1 + 1 + 1
	// 1 + 2
	// 2 + 1
	// 4 = 
	// 1 + 1 + 1 + 1
	// 1 + 1 + 2
	// 1 + 2 + 1 + 1
	// 2 + 1 + 1 + 1
	// 2 + 2

	if (n<=2) return n

	const dp = []
	dp[1] = 1
	dp[2] = 2
	for (let i =3; i <= n; i++) {
		dp[i] = dp[i-1] + dp[i-2]
	}
	return dp[n]
};
```
