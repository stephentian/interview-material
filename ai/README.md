# AI 前端工程师面试资料

> 2024-2025年市场招聘要求整理，涵盖前端AI工程师、前端Agent工程师等岗位

## 岗位概述

### 什么是AI前端工程师？

AI前端工程师是结合传统前端开发能力与AI应用开发能力的新兴岗位，主要负责：

- AI产品的用户界面开发
- LLM（大语言模型）应用的前端集成
- AI Agent可视化交互平台开发
- 流式输出、实时推理结果可视化等技术实现

## 核心技能要求

### 1. 前端基础能力
- 精通 JavaScript/TypeScript/HTML5/CSS3
- 深入理解浏览器工作原理
- 熟练掌握 React/Vue 等主流框架
- 具备良好的工程化能力

### 2. AI应用开发能力
- LLM API 集成与调用
- 流式输出技术（SSE/WebSocket）
- RAG（检索增强生成）系统理解
- Prompt Engineering（提示词工程）
- AI Agent 工作流编排

### 3. 工具与框架
- LangChain / LangGraph
- 向量数据库（Milvus/Pinecone/ChromaDB）
- Embedding 模型应用
- Dify / Coze / n8n 等AI开发平台

## 市场趋势

### 2024-2025年热点
1. **AI Agent** - 从单一对话到多步骤任务执行
2. **RAG应用** - 企业知识库、智能客服等场景
3. **多模态交互** - 文本、图像、语音综合应用
4. **低代码AI平台** - Dify、Coze等平台开发
